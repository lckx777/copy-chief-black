## Conceitos Core

1. **Controle do Processo de Venda**
   - Quem controla o processo, controla a venda
   - Prospect consciente = ele no controle = instável
   - Prospect inconsciente = você no controle = estável

2. **Hierarquia de Influência**
   - Mercado > Oferta > Formato > Mecanismo > Copy
   - Formato esconde venda = menos resistência

3. **Nova Sofisticação: Ângulo do Problema**
   - Quando todos vendem SOLUÇÃO, venda o PROBLEMA
   - Sintoma → Nome → Problema → Descoberta
   - CPC menor, público menos consciente

4. **Diferenças por Gênero**
   - Homem = Status
   - Mulher = Segurança
