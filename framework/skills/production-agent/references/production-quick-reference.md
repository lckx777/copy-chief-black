# Production Quick Reference

> Referência rápida para produção de copy.
> Complementa SKILL.md com checklist executável.

---

## Pre-Flight Checklist

```
[ ] validate_gate research → PASSED
[ ] validate_gate briefing → PASSED
[ ] synthesis.md lido (confidence ≥70%)
[ ] First Principles carregado
```

---

## Workflow Resumido

```
1. GATE: validate_gate (research + briefing)
2. CONTEXT: Ler synthesis.md + summaries
3. FIRST PRINCIPLES: primeiros-principios-copy-chief.md
4. SKILL: Invocar skill apropriado
5. VALIDATE: blind_critic + emotional_stress_test
6. DELIVER: Copy limpa + documentação
```

---

## Thresholds MCP

| Score | Ação |
|-------|------|
| ≥8/10 | APROVADO |
| 6-7.9 | REVISAR |
| <6 | REFAZER |

---

## Modelo 70/30

```
IA: Draft (70%) → Human: Polish (30%) → IA: Variações → Human: Seleção
```

## Anti-Vícios VSL

| Vício IA | Correção |
|----------|----------|
| Leiturabilidade | CONVERSAÇÃO |
| Frases completas | FRAGMENTOS |
| Transições suaves | CORTES ABRUPTOS |
| "Correto" | VISCERAL |
