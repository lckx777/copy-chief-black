# Metodologia de Breakdown

Extrair a ESTRUTURA INVISIVEL de um criativo: intencao psicologica, mecanismos mentais, funcoes estrategicas, formula replicavel.

## 1. Cabecalho Estrategico

Identificar ANTES de analisar:

| Campo | O que identificar |
|-------|-------------------|
| **FORMATO** | UGC, Podcast, POV, Talking Head, Green Screen, Noticia |
| **ANGULO** | Nova Descoberta, Conspiracao, Erro Comum, MUP, MUS |
| **CLUSTER** | Tom + linguagem + dores = publico inferido |
| **PROMESSA** | Transformacao/resultado dominante |
| **CURIOSIDADES** | Gimmick Names, rituais, metodos proprietarios |

## 2. TAG Estrutural

Cada bloco recebe TAG entre [ ] com todos os elementos identificaveis:

```
[Tipo + Dor (especifica) + Gatilho + Mecanismo + Intencao: objetivo]
> Trecho original sem alteracoes...
```

### Componentes

| Componente | Descricao |
|------------|-----------|
| **Tipo** | Funcao estrutural: Gancho, Identificacao, Quebra de objecao, CTA, Transicao |
| **Dor** | Dor ESPECIFICA: "frustracao por reprovar 6 vezes", nao apenas "frustracao" |
| **Gatilho** | Gatilho acionado: identificacao, escassez, urgencia, exclusividade |
| **MUP** | Mecanismo do Problema (quando identificavel) |
| **MUS** | Mecanismo da Solucao (quando identificavel) |
| **MU** | Mecanismo Unico / diferencial proprietario |
| **Intencao** | Objetivo estrategico (OBRIGATORIO em toda TAG) |

Regras: incluir TUDO presente, linguagem ESPECIFICA, MUP/MUS/MU obrigatorios quando identificaveis.

### Analise Tecnica por Bloco

Apos cada TAG + copy, incluir analise tecnica dissecando:
- Por que cada frase funciona (ou nao)
- Qual principio psicologico esta em jogo
- Como se conecta com o bloco anterior/posterior
- Linguagem especifica do avatar presente

## 3. Vocabulario Analitico

Usar como referencia ao etiquetar blocos — nao como obrigacao.

**Dores recorrentes:** frustracao cronica, culpa leve, sensacao de ficar pra tras, vergonha silenciosa, cansaco, inseguranca, injustica percebida, impotencia.

**MUP (causas do problema):** falta de tempo, metodo errado, desinformacao, sistema limitado, crenca de que e tarde, custo sem retorno, complexidade desnecessaria.

**MUS (logica da solucao):** metodo especifico, ferramenta nova, atalho validado, processo simplificado, padrao descoberto, acesso exclusivo.

**MU (diferencial proprietario):** nome memoravel, ritual exclusivo, simbolo unico, comunidade fechada, pesquisa exclusiva.

## 4. Exemplo Worked-Out

**Criativo:** GPT dos Aprovados (Podcast → UGC)

### Cabecalho Estrategico

```
FORMATO: Hibrido: Podcast (Gancho) → UGC/Storytelling (Corpo)
ANGULO: Conspiracao + Nova Descoberta + Transformacao
CLUSTER: Concurseiro CLT, 25-35 anos, 3-7 reprovacoes, frustrado com metodo tradicional
PROMESSA: Aprovacao usando IA com comandos especificos que cursinhos escondem
CURIOSIDADES: (comunidade: Turma do Fundao), (produto: Comandos dos Aprovados), (vilao: cursinho "E"), (gatilho: "quase preso")
```

### Extracao Estrutural

**BLOCO 1 — HOOK PODCAST**

```
[Pattern Interrupt + Noticia Chocante + Qualificacao (TJ-SP) + Autoridade Implicita (G1) + Intencao: capturar atencao de concurseiros e gerar curiosidade instantanea]
> A: Mano, voce viu a historia do cara que quase foi preso por usar ChatGPT pra passar no TJ de Sao Paulo? Saiu ate no G1.
```

Analise tecnica:
- "Mano" = tom coloquial, linguagem de WhatsApp entre concurseiros
- "quase foi preso" = choque emocional extremo, ativa medo + curiosidade
- "TJ de Sao Paulo" = especifico, qualifica avatar concurseiro
- "Saiu ate no G1" = autoridade externa, parece noticia real

**BLOCO 2 — AMPLIFICACAO**

```
[Pergunta Retorica + Loop Aberto + Gatilho de Medo + Intencao: manter tensao e amplificar curiosidade com segunda voz]
> B: Perai. Preso? Por usar ChatGPT? Isso e crime agora?
```

Analise tecnica:
- Segunda voz = verbaliza objecao do prospect (ele pensa exatamente isso)
- "Isso e crime agora?" = medo + confusao = atencao maxima
- Estrutura de podcast legitima a conversa como "real"

**BLOCO 3 — TRANSICAO**

```
[Revelacao Parcial + Setup do Vilao + Transicao Corpo + Loop Mantido + Intencao: transformar curiosidade em indignacao]
> A: Nao e crime. O problema foi o que o cursinho fez depois. O cara gravou um video contando tudo, olha so...
```

Analise tecnica:
- "Nao e crime" = alivia tensao parcialmente (respiro)
- "o que o cursinho fez depois" = vilao aparece pela primeira vez
- "olha so" = transicao natural pra UGC
- Loop permanece aberto → corpo vai resolver

**BLOCO 4 — IDENTIFICACAO**

```
[Identificacao + Dor (6 reprovacoes + falta de tempo + memoria ruim) + Gatilho de Pertencimento + MUP implicito (metodo tradicional) + Intencao: gerar empatia visceral]
> Eu ja tinha sido reprovado mais de seis vezes nesse concurso. Nunca fui bom de memoria, trabalhava o dia inteiro e tinha pouco tempo para estudar.
```

Analise tecnica:
- "6 vezes" = numero especifico (nao redondo), crivel
- "Nunca fui bom de memoria" = normaliza limitacao do avatar
- "trabalhava o dia inteiro" = confirma avatar CLT
- "pouco tempo" = dor universal do concurseiro

**BLOCO 5 — VILAO**

```
[Vilao Estabelecido + Protecao Juridica + Gatilho de Indignacao + Dor (R$2.000 desperdicados) + Intencao: criar antagonista claro sem exposicao legal]
> Como quase todo concurseiro nessa situacao, resolvi me matricular em um cursinho preparatorio famoso no Brasil. Por motivos judiciais, eu nao posso citar o nome, mas ele comeca com a letra "E". Gastei mais de dois mil reais em aulas, PDFs e simulados, e mesmo assim continuava travado.
```

Analise tecnica:
- "comeca com a letra E" = todo mundo entende, gera cumplicidade
- "Por motivos judiciais" = drama + credibilidade
- "mais de dois mil reais" = dor financeira especifica
- "continuava travado" = frustracao cronica

**BLOCO 6 — PARADOXO**

```
[Paradoxo + Gatilho de Curiosidade + Quebra de Crenca + Intencao: dissonancia cognitiva]
> Foi ai que eu comecei a perceber algo estranho. Eu via pessoas na mesma situacao que a minha, sem dominar todo o conteudo, sem estudar o dia inteiro, sendo aprovadas. Aquilo nao fazia sentido.
```

Analise tecnica:
- "algo estranho" = inicio do misterio
- "mesma situacao" = reforco de identificacao
- "nao fazia sentido" = dissonancia = prospect PRECISA de explicacao

**BLOCO 7 — DESCOBERTA**

```
[Jornada de Descoberta + MU (Turma do Fundao) + Gatilho de Exclusividade + Prova Social Coletiva + Intencao: revelar comunidade secreta]
> Entao eu comecei a investigar o que essas pessoas estavam fazendo de diferente. Foi assim que eu encontrei uma comunidade no Facebook chamada "Turma do Fundao". La tinham varios relatos de gente comum sendo aprovada em concursos dificeis.
```

Analise tecnica:
- "Turma do Fundao" = Gimmick Name memoravel
- "comunidade no Facebook" = especifico + acessivel
- "gente comum" = prospect se ve ali
- "concursos dificeis" = amplifica valor da descoberta

**BLOCO 8 — MECANISMO**

```
[MUS Completo + 4 Funcoes Especificas + Gatilho de Clareza + Intencao: mostrar COMO funciona]
> Muitos usavam inteligencia artificial, mas com comandos especificos que faziam o ChatGPT gratuito analisar o edital, identificar padroes da banca, priorizar o que realmente cai e gerar resumos e questoes no nivel da prova.
```

Analise tecnica:
- "nao estudavam tudo" = contraintuitivo
- 4 funcoes concretas: analisar edital, identificar padroes, priorizar, gerar material
- "ChatGPT gratuito" = remove objecao de preco
- Especificidade do MUS gera confianca

**BLOCO 9 — PROVA + CTA 1**

```
[Prova Social Pessoal + Resultado Tangivel + CTA Suave + Intencao: validar metodo e criar primeiro ponto de conversao]
> Depois que comecei a utilizar esses comandos especificos, eu finalmente consegui a minha aprovacao. Eu ate deixei esses comandos disponiveis para voces no botao aqui embaixo.
```

Analise tecnica:
- "finalmente" = jornada longa, vitoria merecida
- CTA suave ANTES do drama final = duplo ponto de conversao

**BLOCO 10 — DRAMA DO VILAO**

```
[Vilao em Acao + Injustica + Gatilho de Raiva + Conspiracao + Intencao: transformar historia em cruzada pessoal]
> Depois da aprovacao, o cursinho quis usar a minha imagem para fazer propaganda, como se o merito fosse deles. Eu recusei. A partir dai comecaram as ameacas juridicas. Na epoca, eu quase fui preso. Eles alegaram que eu estava envolvido em esquema de gabarito vazado. Isso me deixa puto ate hoje.
```

Analise tecnica:
- "como se o merito fosse deles" = indignacao justificada
- "quase fui preso" = fecha loop do hook
- "Isso me deixa puto ate hoje" = emocao crua, autentica

**BLOCO 11 — CTA FINAL**

```
[Reason Why Price + Escassez Real + CTA Final + Urgencia Implicita + Intencao: justificar acao imediata]
> Se eu tivesse conhecido esses "Comandos dos Aprovados" antes, nem teria gasto mais de dois mil reais com o cursinho. Infelizmente, a comunidade "Turma do Fundao" saiu do ar. Mas antes disso acontecer, eu salvei em um PDF todos os comandos. Hoje esse material esta organizado dentro do "GPT dos Aprovados". Eu acho que ainda esta disponivel.
```

Analise tecnica:
- "R$2.000 vs comandos" = contraste de preco implicito
- "saiu do ar" = escassez REAL, nao fake
- "Eu acho que ainda esta disponivel" = urgencia soft, natural

### Mapa Estrutural

1. [PODCAST HOOK] (pattern interrupt) → 2. [AMPLIFICACAO] (segunda voz) → 3. [TRANSICAO] (vilao teased) → 4. [IDENTIFICACAO] (avatar se reconhece) → 5. [VILAO] (cursinho "E") → 6. [PARADOXO] (dissonancia) → 7. [DESCOBERTA] (comunidade secreta) → 8. [MECANISMO] (MUS revelado) → 9. [PROVA + CTA 1] (aprovacao) → 10. [DRAMA] (vilao ataca) → 11. [CTA FINAL] (escassez natural)

### Formula Invisivel

HOOK DE NOTICIA CHOCANTE → IDENTIFICACAO VISCERAL → VILAO NOMEADO → PARADOXO → COMUNIDADE SECRETA → MECANISMO REVELADO → PROVA PESSOAL → DRAMA DO HEROI → ESCASSEZ NATURAL → CTA COM URGENCIA IMPLICITA

### Tecnicas-Chave

| Tecnica | Funcao | Execucao no Criativo |
|---------|--------|----------------------|
| Podcast Hook | Pattern interrupt + conversa real | 2 vozes discutindo "noticia" |
| Vilao Oculto | Raiva sem exposicao juridica | "Comeca com a letra E" |
| Comunidade Secreta | Exclusividade + FOMO + Prova Social | "Turma do Fundao" |
| Paradoxo | Dissonancia cognitiva | "Sem estudar tudo, sendo aprovadas" |
| Drama do Heroi | Empatia + indignacao | Ameacas juridicas apos aprovacao |
| Escassez Natural | Urgencia sem fake | "Comunidade saiu do ar" |
| Duplo CTA | Duas chances de conversao | CTA pos-aprovacao + CTA final |

### Analise NUUPPECC

| Criterio | Presente | Execucao |
|----------|----------|----------|
| Novo | ✅ | Comandos especificos por banca |
| Urgente | ✅ | "Acho que ainda esta disponivel" |
| Unico | ✅ | "Turma do Fundao", "Comandos dos Aprovados" |
| Paradoxal | ✅ | "Quase preso por usar ChatGPT" |
| Pessoal | ✅ | Historia em primeira pessoa, 6 reprovacoes |
| Especifico | ✅ | TJ-SP, 6 vezes, R$2.000, letra "E" |
| Concreto | ✅ | 4 funcoes do MUS detalhadas |
| Controverso | ✅ | Cursinhos escondem IA |

### Por Que Funciona

1. **Hook Hibrido** — Podcast da credibilidade de "conversa real", transicao natural pra storytelling
2. **Jornada do Heroi completa** — Mundo Comum → Chamado → Mentor → Provacao → Transformacao → Vilao Final
3. **Vilao Triplice** — Cursinhos caros + Metodo tradicional + Sistema corrupto
4. **Escassez Real** — "Turma do Fundao saiu do ar" + "Acho que ainda esta disponivel"
5. **MUS com 4 funcoes concretas** — Analisar edital, identificar padroes, priorizar, gerar material
