# Task: Headline Testing Framework

> **Primary Expert:** John Caples
> **Supporting:** Eugene Schwartz (awareness), David Ogilvy (long headlines)

---

## Process

### Step 1: Diagnose (Schwartz)
- Awareness level: ___
- Sophistication level: ___
- DRE: ___

### Step 2: Generate 10+ Variations (Caples Formulas)

| # | Formula | Headline | Awareness Match |
|---|---------|----------|----------------|
| 1 | How-To | "How to [result]..." | |
| 2 | Number | "[X] ways to [benefit]..." | |
| 3 | Question | "Do you [problem]?" | |
| 4 | Command | "[Verb] your [thing]" | |
| 5 | News | "Announcing [discovery]..." | |
| 6 | Testimonial | "[Quote]" | |
| 7 | If-Then | "If you [situation], then..." | |
| 8 | Long (Ogilvy) | "[Benefit + detail + mechanism]" | |
| 9 | Curiosity | "[Unexpected] + [specific detail]" | |
| 10 | Challenge | "[Common belief] is wrong. Here's why." | |

### Step 3: Score Each Headline

| Criterion | Weight | H1 | H2 | H3 | H4 | H5 |
|-----------|--------|----|----|----|----|-----|
| Benefit clarity | 25% | | | | | |
| Curiosity/open loop | 25% | | | | | |
| Specificity | 20% | | | | | |
| Awareness match | 15% | | | | | |
| Uniqueness (Logo Test) | 15% | | | | | |
| **TOTAL** | 100% | | | | | |

### Step 4: Test Plan

```
TEST A vs B: Top 2 scored headlines
METRIC: CTR (for ads) or read-through rate (for VSL/LP)
SAMPLE: Minimum 1,000 impressions per variation
DURATION: 48-72 hours minimum
DECISION: Winner by 95% statistical significance
```

## Quality Checklist

- [ ] 10+ variations generated
- [ ] Multiple formulas used (not just How-To)
- [ ] Awareness level diagnosed BEFORE writing
- [ ] Each headline includes a specific benefit
- [ ] Top 3 selected by scoring matrix, not gut feeling
- [ ] Test plan defined with success metrics
- [ ] Logo Test passed on all finalists
