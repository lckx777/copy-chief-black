# Task: Logo Test Verification

> **Primary Expert:** David Ogilvy (brand-specific advertising)
> **Supporting:** Todd Brown (mechanism differentiation)

---

## Process

### Step 1: Competitor Swap Test
For each section of copy, ask:
"Could [Competitor A] use this WITHOUT changing it?"

| Section | Swappable? | Why/Why Not |
|---------|-----------|-------------|
| Headline | YES/NO | |
| Lead | YES/NO | |
| Problem | YES/NO | |
| Mechanism | YES/NO | |
| Solution | YES/NO | |
| Proof | YES/NO | |
| Offer | YES/NO | |
| CTA | YES/NO | |

### Step 2: Identify Weak Points
List every "YES" (swappable) section. These MUST be rewritten.

### Step 3: Add Differentiators
For each weak section:
- Add proprietary mechanism name: ___
- Add specific proof (name, number, date): ___
- Add unique origin story element: ___
- Add brand-specific language: ___

### Step 4: Re-Test
Repeat Step 1 after changes. All sections must be "NO" (not swappable).

## Quality Checklist

- [ ] Every section tested individually
- [ ] Zero "YES" sections remaining
- [ ] Proprietary mechanism name present in 3+ sections
- [ ] Specific numbers appear (non-round)
- [ ] Origin story referenced
- [ ] Copy is unmistakably THIS offer
