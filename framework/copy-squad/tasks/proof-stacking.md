> **Primary Expert:** Claude Hopkins (scientific advertising)
> **Supporting:** David Ogilvy (factual approach), Gary Bencivenga (proof)

## Process

### Step 1: Inventory Available Proof

| Type | Available? | Strength (1-10) | Source |
|------|-----------|-----------------|--------|
| Clinical study | | | |
| Expert endorsement | | | |
| User testimonials | | | |
| Before/after data | | | |
| Media mentions | | | |
| Patent/certification | | | |
| Mechanism explanation | | | |
| Comparison data | | | |
| Guarantee/warranty | | | |
| Third-party audit | | | |

### Step 2: Stack in Persuasion Order
1. **Mechanism Proof** (WHY it works — credibility)
2. **Authority Proof** (WHO backs it — trust)
3. **Social Proof** (WHO uses it — safety)
4. **Results Proof** (WHAT happens — desire)
5. **Risk Reversal** (guarantee — action)

### Step 3: Write Each Proof Block
For each proof, include:
- The claim it supports
- The specific evidence
- The emotional reaction it should trigger

## Quality Checklist

- [ ] Minimum 5 proof types used
- [ ] Stacked in correct persuasion order
- [ ] Each proof has specific numbers
- [ ] No generic "thousands of customers"
- [ ] Proof supports specific claims (not floating)
- [ ] Risk reversal closes the stack
