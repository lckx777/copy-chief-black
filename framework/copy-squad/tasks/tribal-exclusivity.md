# Task: Tribal Exclusivity Activation

> **Primary Expert:** Robert Cialdini (scarcity + social proof)
> **Supporting:** Russell Brunson (tribe building), Alex Hormozi (value equation)

## Process

### Step 1: Define In-Group vs Out-Group

| Element | In-Group (Buyers) | Out-Group (Non-Buyers) |
|---------|-------------------|----------------------|
| Label | "People who ___" | "People who still ___" |
| Behavior | ___ | ___ |
| Belief | ___ | ___ |
| Language | ___ | ___ |

### Step 2: Create Exclusivity Markers
- Secret knowledge: "What [authority] discovered..."
- Limited access: "Only [X] people know about..."
- Early adopter: "Before [mainstream] catches on..."
- Insider language: "[Gimmick Name]"

### Step 3: Social Proof Stack
- Quantity: "[X] already using..."
- Quality: "[Specific person] achieved [specific result]..."
- Speed: "In just [specific timeframe]..."
- Variety: "[Professional], [Parent], [Student]..."

## Quality Checklist

- [ ] Clear in-group/out-group distinction
- [ ] Exclusivity doesn't feel manipulative
- [ ] Social proof uses specific numbers (non-round)
- [ ] Multiple proof types (quantity + quality + speed)
- [ ] Language matches VOC
