# Task: Lead Generation / Lead Magnet

> **Primary Expert:** Gary Halbert (hook + specificity), Robert Collier (entry)
> **Supporting:** Frank Kern (results in advance), Eugene Schwartz (awareness)

---

## Lead Magnet Types by Awareness

| Awareness | Best Lead Magnet | Why |
|-----------|-----------------|-----|
| Unaware | Quiz / Assessment | Reveals a problem they didn't know they had |
| Problem Aware | Guide / Checklist | "Here's why [symptom] happens" |
| Solution Aware | Comparison / Case Study | "Here's which approach actually works" |
| Product Aware | Free Trial / Demo | "See it work for yourself" |

## Structure

```
1. HOOK (Halbert)     - Curiosity + specificity in the headline
2. ENTRY (Collier)    - Meet them where they are mentally
3. VALUE (Kern)       - Deliver a real result in advance
4. BRIDGE             - Connect the free result to the paid offer
5. CTA                - Simple next step to continue the journey
```

## Lead Magnet Quality Criteria

| Criterion | Check |
|-----------|-------|
| Delivers ACTUAL result (not just information) | [ ] |
| Specific to ONE problem (not comprehensive) | [ ] |
| Consumable in under 15 minutes | [ ] |
| Creates desire for the paid offer naturally | [ ] |
| Title uses curiosity + specificity | [ ] |
| Demonstrates expertise through RESULT | [ ] |
| Landing page passes Halbert specificity test | [ ] |

## Quality Checklist

- [ ] Awareness level diagnosed
- [ ] Lead magnet type matches awareness
- [ ] Title has Halbert-level specificity
- [ ] Opens by entering the reader's conversation (Collier)
- [ ] Delivers a genuine quick win (Kern results-in-advance)
- [ ] Natural bridge to paid offer (no hard pivot)
- [ ] Opt-in page has 3+ specificity elements
- [ ] Follow-up sequence planned (Chaperon soap opera)
