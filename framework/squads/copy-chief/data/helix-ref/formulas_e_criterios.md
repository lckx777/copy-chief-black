# Fórmulas e Critérios do HELIX System

## Fórmulas Principais

### One Belief Statement (Fase 10)

```
"[NOVA OPORTUNIDADE] é a chave para [DESEJO] e só é alcançável através de [MEU MECANISMO]."
```

Componentes:
- Nova Oportunidade: O que é diferente do que já tentaram (2-4 palavras)
- Desejo: O resultado final que buscam (específico e emocional)
- Meu Mecanismo: Por que só funciona através do meu método

Teste de validação: Pergunte "Se acreditar nisso, VAI comprar?" — Se sim, está correto.

### Gancho da Solução (Fases 6/9)

```
"Já ouviu falar desse [GIMMICK NAME] que [ORIGEM] estão usando secretamente para [DESEJO]? Já estão chamando isso de [AUTHORITY HOOK]."
```

Exemplo: "Já ouviu falar desse 'Sal Marinho Japonês' que médicos de Okinawa estão usando secretamente para energia ilimitada? Já estão chamando isso de 'O Mineral Esquecido'."

### 4 Camadas do Mecanismo

Camada 1 - Hero Ingredient: Componente principal que resolve (ex: Sal marinho com 84 minerais)
Camada 2 - Gimmick Name: Nome chiclete memorável (ex: "Protocolo do Sal Ancestral")
Camada 3 - Origin Story: De onde veio, credibilidade (ex: Descoberto em vila de centenários)
Camada 4 - Authority Hook: Gancho de autoridade derivado (ex: "O segredo dos centenários japoneses")

## Critérios de Validação

### 3 Critérios RMBC (MUP/MUS)

Todo mecanismo único deve passar nos 3:

<criterio_1>
Facilmente digerível e transferível?
- Consegue explicar em 1-2 frases simples?
- A pessoa conseguiria contar para outra pessoa?
- Usa linguagem do dia-a-dia?
</criterio_1>

<criterio_2>
Genuinamente interessante e único?
- Gera reação "wow" ou "não sabia disso"?
- Diferente do que já ouviram?
- Desperta curiosidade?
</criterio_2>

<criterio_3>
Intuitivamente E empiricamente provável?
- Faz sentido lógico? (intuição)
- Tem alguma base real? (empírico)
- Não parece "bom demais para ser verdade"?
</criterio_3>

Se falhar em qualquer critério: Refine até passar nos 3.

### 6-Question Litmus Test (Headlines/Leads)

Cada headline/lead deve passar em pelo menos 4:

1. Oferece RECOMPENSA por ler? (Verifica benefício claro)
2. Tem ESPECÍFICOS que intrigam? (Verifica números, detalhes)
3. Dispara EMOÇÃO FORTE existente? (Verifica conexão com DRE)
4. Tem elemento de INTRIGA? (Verifica curiosidade, mistério)
5. É CLARA e simples? (Verifica se não é confusa)
6. Conecta com EVENTOS ATUAIS? (Verifica relevância temporal)

## Quality Gates por Fase

### Gate Fase 1 (Identificação)
- Mercado + Subnicho claramente definidos
- Necessidade humana identificada (das 6)
- Nome chiclete validado (curiosidade + prova implícita)

### Gate Fase 3 (Avatar)
- DRE identificada com manifestações específicas
- Linguagem exata coletada (frases copy-paste)
- 5 Medos mapeados com cenários específicos

### Gate Fase 5 (MUP)
- Vilão com nome chiclete
- Passa nos 3 critérios RMBC
- Metáfora/analogia clara

### Gate Fase 6 (MUS)
- Conexão lógica MUP -> MUS evidente
- 4 Camadas preenchidas (Hero, Gimmick, Origin, Authority)
- Arsenal de provas por categoria

### Gate Fase 10 (One Belief)
- Fórmula preenchida corretamente
- Passa no teste: "Se acreditar, VAI comprar?"
- 10 Perguntas com respostas prontas para VSL

## Teste Anti-Superficialidade

Aplique em CADA campo preenchido:

Especificidade: Há números, prazos, exemplos concretos?
Emoção: Consigo visualizar a pessoa sentindo isso?
Prova: Existe caminho de prova para essa afirmação?
Diferenciação: Serviria para o concorrente? (Se sim, refazer)
Acionabilidade: Copywriter consegue escrever só lendo isso?

Teste Final: "Um copywriter júnior conseguiria escrever uma VSL de alta conversão apenas seguindo este briefing?"
- SIM = Briefing aprovado
- TALVEZ ou DEPENDE = Briefing superficial, refazer

## Conexão MUP <-> MUS (Espelhamento)

O MUS DEVE ser a solução lógica e direta para o MUP.

Exemplos de espelhamento correto:
- MUP: Toxinas bloqueiam receptores -> MUS: Remove toxinas, desbloqueia receptores
- MUP: Inflamação crônica silenciosa -> MUS: Anti-inflamatório natural de alta absorção
- MUP: Hormônio X em baixa desde Y -> MUS: Estimula produção natural de hormônio X
- MUP: Bactéria Z domina intestino -> MUS: Mata bactéria Z, recoloniza com probióticos

Erro comum: MUP fala de A, MUS fala de B. Devem ser ESPELHOS.
