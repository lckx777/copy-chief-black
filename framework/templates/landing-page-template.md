## 14 BLOCOS PERSUASIVOS

### BLOCO 1: HEADER + HEADLINE

**Objetivo:** Capturar atenção em 3 segundos.

**Fluxo Emocional:**
- Entrada: Neutro/Scroll (prospect chegou via ad)
- Saída: Curiosidade (headline gerou interesse)
- DRE Level: 1

**Elementos:**
- Logo (se aplicável)
- Headline principal (Big Promise)
- Sub-headline (qualificação ou curiosidade)

```
[HEADLINE]

[Sub-headline]
```

---

### BLOCO 2: VÍDEO/LEAD (se VSL/Hybrid)

**Objetivo:** Engajar com conteúdo principal.

**Fluxo Emocional:**
- Entrada: Curiosidade (headline gerou interesse)
- Saída: Engajamento (conteúdo prendeu atenção)
- DRE Level: 1

**Para VSL Page:**
- Player de vídeo centralizado
- Thumbnail estratégica
- CTA abaixo do vídeo

**Para TSL Page:**
- Lead copy expandido
- Hook textual forte

```
[Área do vídeo OU lead copy]
```

---

### BLOCO 3: PROBLEMA AGITADO

**Objetivo:** Validar a dor do prospect.

**Fluxo Emocional:**
- Entrada: Engajamento (prospect está atento)
- Saída: Reconhecimento + Medo (se viu no problema)
- DRE Level: 2

**Estrutura:**
- "Você já [situação frustrante]?"
- Lista de sintomas/dores (bullet points)
- Sinestesia emocional

```
[Desenvolver agitação do problema]
```

---

### BLOCO 4: VILÃO REVELADO

**Objetivo:** Externalizar culpa, criar inimigo comum.

**Fluxo Emocional:**
- Entrada: Medo (problema é real e sério)
- Saída: Raiva/Indignação (culpa externalizada)
- DRE Level: 3

**Vilão:** [extrair de Fase 05]

```
[Revelar o vilão de forma impactante]
```

**Fluxo Emocional:**
- Entrada: Raiva (indignação com o vilão)
- Saída: Curiosidade + Esperança (existe uma descoberta)
- DRE Level: 2

**MUP:** [descriir de Fase 05]

```
[Apresentar a descoberta/mecanismo]
```

- Entrada: Esperança (mecanismo faz sentido)
- Saída: Crença (solução é legítima)
- DRE Level: 2

**MUS:** [extrair de Fase 06]

**Elementos:**
- Nome do produto
- O que é (1-2 frases)

```
[Apresentar a solução]
```

### BLOCO 7: BENEFÍCIOS + TRANSFORMAÇÃO

**Objetivo:** Pintar o futuro desejado.

**Fluxo Emocional:**

- Saída: Desejo (quer a transformação)
- DRE Level: 3

**Estrutura:**
- Lista de benefícios (features → benefits)
- Antes vs Depois
- Promessa de transformação

| Antes | Depois |
|-------|--------|
| [dor] | [resultado] |
| [dor] | [resultado] |
| [dor] | [resultado] |

```
[Desenvolver benefícios e transformação]
```

### BLOCO 8: PROVA SOCIAL

**Objetivo:** Eliminar ceticismo com resultados reais.

**Fluxo Emocional:**
- Entrada: Desejo (quer mas ainda duvida)
- Saída: Confiança (outros conseguiram, eu também)
- DRE Level: 2

**Tipos de prova:**
- [ ] Depoimentos em vídeo
- [ ] Depoimentos em texto
- [ ] Screenshots de resultados
- [ ] Números/estatísticas
- [ ] Logos de mídia (se aplicável)

```
[Organizar provas sociais]
```

### BLOCO 11: GARANTIA

**Objetivo:** Eliminar risco percebido.

**Fluxo Emocional:**
- Entrada: Medo de Perder (investimento é real)
- Saída: Segurança (risco zero para mim)
- DRE Level: 1

**Tipo:** [Incondicional / Condicional / Dupla]
**Prazo:** [X] dias

```
[Desenvolver garantia de forma confiante]
```

---

### BLOCO 12: PREÇO + CTA PRINCIPAL

**Objetivo:** Apresentar investimento e direcionar ação.

**Fluxo Emocional:**
- Entrada: Segurança (garantia removeu risco)
- Saída: Urgência (preciso agir agora)
- DRE Level: 4

**Elementos:**
- Âncora de preço (valor total do stack)
- Preço real
- Economia/desconto
- Botão de CTA claro
- Formas de pagamento

```
De R$ [âncora]
Por apenas R$ [preço]
ou [parcelas] de R$ [valor]

[BOTÃO CTA]

[Selos de segurança]
```

---

### BLOCO 13: FAQ

**Objetivo:** Resolver objeções finais.

**Fluxo Emocional:**
- Entrada: Dúvidas residuais (objeções finais)
- Saída: Segurança (todas as dúvidas respondidas)
- DRE Level: 1

**Objeções comuns a responder:**
- [ ] "Funciona pra mim?"
- [ ] "Quanto tempo até ver resultado?"
- [ ] "E se eu não gostar?"
- [ ] "Como acesso?"
- [ ] "Preciso de [X] para funcionar?"

```
**Pergunta 1?**
Resposta...

**Pergunta 2?**
Resposta...
```

---

### BLOCO 14: CTA FINAL + URGÊNCIA

**Objetivo:** Último empurrão para ação.

**Fluxo Emocional:**
- Entrada: Segurança (objeções resolvidas)
- Saída: Ação (decisão de compra)
- DRE Level: 4-5

**Elementos:**
- Resumo da transformação
- Escassez/urgência (se real)
- CTA repetido
- Consequência de não agir

```
[Desenvolver fechamento com urgência]

[BOTÃO CTA FINAL]
```
