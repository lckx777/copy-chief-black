## FORMATO vs ÂNGULO (Distinção Crítica)

### FORMATO (Embalagem Visual)
O que o espectador VÊ:

| Formato | Descrição | Quando Usar |
|---------|-----------|-------------|
| UGC | Pessoa comum falando para câmera | Prova social, relatabilidade |
| Talking Head | Expert/autoridade apresentando | Credibilidade, educação |
| React | Reação a conteúdo | Entretenimento, viralidade |
| POV | Perspectiva primeira pessoa | Imersão, storytelling |
| Tela Dividida | Comparação lado a lado | Antes/depois, vs |
| Green Screen | Apresentador + imagem fundo | Explicação, contexto |
| Screencast | Gravação de tela | Tutorial, demonstração |
| B-Roll + VO | Imagens + voz off | Storytelling visual |

### ÂNGULO (Abordagem da Mensagem)
COMO a copy é passada:

| Ângulo | Estrutura | Exemplo de Hook |
|--------|-----------|-----------------|
| Nova Descoberta | "Cientistas descobriram..." | "Novo estudo revela que..." |
| Erro Comum | "O erro que 90% comete..." | "Pare de fazer isso agora..." |
| Paradoxo | "Como X sem Y..." | "Emagreça comendo mais..." |
| Conspiração | "O que eles escondem..." | "A indústria não quer que..." |
| Curiosidade | "Você sabia que..." | "3 coisas sobre X que..." |
| Urgência | "Última chance..." | "Antes que acabe..." |
| Social Proof | "10.000 pessoas já..." | "Veja o que Maria descobriu..." |
| Contrarian | "Esqueça tudo que..." | "Por que X não funciona..." |

## ESTRUTURA DO CRIATIVO

### HOOK (0-3 segundos)

**Objetivo:** Parar o scroll. Criar loop aberto impossível de ignorar.

**Fórmula:** [Ângulo] + [Promessa específica] + [Qualificador]

**Tipos de Hook:**
1. **Pergunta provocativa:** "Você sabia que [fato chocante]?"
2. **Afirmação bold:** "[Resultado específico] em [tempo curto]"
3. **Erro revelado:** "O erro de R$[X] que você comete todo dia"
4. **Paradoxo:** "Como [resultado] sem [sacrifício esperado]"
5. **Gatilho emocional:** "[Emoção intensa] quando [situação]"

**Hook escolhido:**
```
[Escrever hook - máximo 2 linhas]
```

**Variações (mínimo 3):**
```
Variação A: [hook alternativo]
Variação B: [hook alternativo]
Variação C: [hook alternativo]
```

---

### BODY (3-45 segundos)

**Objetivo:** Desenvolver interesse, preparar para CTA.

**Estrutura recomendada:**

| Tempo | Elemento | Conteúdo |
|-------|----------|----------|
| 3-10s | Problema | Agitar dor específica |
| 10-20s | Mecanismo | Revelar insight/descoberta |
| 20-35s | Solução | Apresentar oferta |
| 35-45s | Prova | Resultado/depoimento rápido |

**Body:**
```
[Desenvolver corpo do criativo]
```

---

### CTA (últimos 5-10 segundos)

**Objetivo:** Direcionar ação clara.

**Fórmula:** [Ação] + [Urgência] + [Benefício]

**Exemplos:**
- "Clique no link e comece hoje mesmo"
- "Toque em 'saiba mais' antes que [urgência]"
- "Acesse o link na bio e descubra [benefício]"

**CTA escolhido:**
```
[Escrever CTA]
```
