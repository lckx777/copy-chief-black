# Revisão Universal - Checklists Pré-Entrega v5

> **Aplicar em:** TODA copy antes de entregar.
> Não entregar NADA sem passar por estes checklists.
> Fundamentação: Metodologia Vitor Mound + Fundamentos Universais v5

---

## Framework de Revisão

### O Princípio

Após escrever, espere pelo menos 30 minutos. Volte e leia como se fosse um lead cético, apressado, desconfiado. Critique impiedosamente.

---

## Checklist 1: Hook

Antes de finalizar, verificar:

```
[ ] O hook QUALIFICA o público-alvo?
[ ] O hook transmite uma EMOÇÃO específica?
[ ] O hook gera CURIOSIDADE?
[ ] A informação mais importante vem PRIMEIRO na frase?
[ ] Existe NOME CHICLETE (se aplicável)?
[ ] Existe FRASE DE PODER (se aplicável)?
[ ] O hook faria alguém PARAR de rolar o feed?
```

### Teste Final do Hook

> "Eu pararia para ver isso no meu feed?"

Se a resposta for NÃO → Reescreva, mesmo que signifique "quebrar" a estrutura.

---

## Checklist 2: Body

```
[ ] Existem DISPAROS DE DOPAMINA após seções técnicas?
[ ] O texto é CONCISO (não prolixo)?
[ ] As afirmações são ESPECÍFICAS (não vagas)?
[ ] Os ELEMENTOS ESSENCIAIS estão presentes?
    - Benefício principal ✓
    - Future pacing ✓
    - Pontos de dor ✓
    - Mecanismo diferenciado ✓
    - Prova ✓
[ ] O texto FLUI naturalmente quando lido em voz alta?
[ ] Existe PROVA adequada?
```

### Teste de Prolixidade

Para cada parágrafo, pergunte: "Posso falar isso de forma mais curta?"
Se sim → Reescreva.

**Meta:** Tirar 5-10% do texto. Se for impossível tirar algo, a copy está boa.

---

## Checklist 3: Congruência

```
[ ] Avatar é adequado para o público-alvo?
[ ] Cenário é congruente com a mensagem?
[ ] Tom de comunicação combina com o avatar?
[ ] Elementos visuais PROVAM o que o texto diz?
[ ] Edição é congruente com o tipo de comunicação?
```

---

## Checklist 4: Documento para Editor

```
[ ] Link do avatar/ator está incluído?
[ ] Referências de edição (links) estão incluídas?
[ ] Comentários explicativos estão claros?
[ ] Músicas/sons desejados estão indicados?
[ ] Tom de voz está descrito?
[ ] Um editor conseguiria recriar sua visão?
```

---

## Checklist 5: Anti-Prolixidade

| Trecho | Pode encurtar? | Ação |
|--------|----------------|------|
| [verificar cada parágrafo] | Sim/Não | Cortar/Manter |

**Regra:** Se está longo demais, está errado.

---

## Checklist 6: Especificidade = Cena de Filme (v5.0)

Para cada afirmação importante, verificar AMBAS as faces:

### Face DATA (números, nomes, datas)

| Pergunta | Vago (❌) | Específico (✅) |
|----------|----------|----------------|
| Quanto? | "Muito" | "12,4kg" |
| Em quanto tempo? | "Algumas semanas" | "47 dias" |
| Quem disse? | "Estudos comprovam" | "Harvard 2023" |
| Quando? | "Recentemente" | "15 de março" |
| Onde? | "Uma universidade" | "MIT, Boston" |
| Como? | "Tomando o chá" | "200ml às 7h, em jejum" |

### Face NARRATIVA (cena de filme - MAIS PODEROSA)

| De (DATA) | Para (NARRATIVA) |
|-----------|------------------|
| "Estudei 4 anos" | "Em 6 anos pagando cursinho, ninguém me ensinou isso" |
| "89% reprovam" | "Uma palavra me custou a vaga na PF" |
| "Perdi 12kg" | "Minha esposa achou que eu estava doente" |
| "Pressão 140/90" | "Acordei 3AM suando frio, coração disparado" |

**Pergunta Final:** Consigo VER O FILME ou só ler dados?

## Perguntas Obrigatórias de Revisão (v5.0 - ATUALIZADO)

Antes de entregar, responda:

1. **Eu pararia para ver isso no feed?**
2. **Isso está chato?**
3. **Essa frase é necessária?**
4. **Estou sendo prolixo aqui?**
5. **O hook está forte o suficiente?**
6. **Existe alguma parte que eu pularia se estivesse assistindo?**
7. **NOVO v5:** DRE identificada e níveis 4-5 ativados?
8. **NOVO v5:** Especificidade tem NARRATIVA (não só DATA)?
9. **NOVO v5:** Tom calibrado pelo briefing (não hardcodado)?
10. **NOVO v5:** Coerência interna mantida (detalhes não se contradizem)?

Se a resposta a qualquer uma for "sim" (exceto a primeira) → Reescreva.

---

## Processo de Revisão

```
1. Escrever a copy
2. Esperar 30+ minutos
3. Reler como consumidor cético
4. Rodar os 6 checklists acima
5. Fazer ajustes necessários
6. Reler uma última vez em voz alta
7. Só então entregar
```

---

## O Erro que Todo Mundo Comete

Copywriters não releem o próprio trabalho com olhar crítico. Não se colocam no lugar do lead.

**A solução:**
- Ler em voz alta
- Ler como se fosse um lead cético, apressado, desconfiado
- Criticar impiedosamente ANTES de enviar
