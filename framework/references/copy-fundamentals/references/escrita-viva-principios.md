# Escrita Viva: Princípios Universais v5

> **REGRA:** Estes são FUNDAMENTOS que SEMPRE se aplicam, independente de nicho, oferta ou avatar.
> NÃO confundir com ESTILO (tom, emoção, gírias) que varia por contexto.
> **NOVO v5:** Tom não é fixo. Calibrado pelo briefing e DRE.

---

## 1. Tensão Dramática

### O que é
Conflito emocional que mantém o leitor engajado. Não é apenas "problema + solução", é DRAMA.

### Princípio Universal
Toda copy precisa de pelo menos UM elemento de tensão:
- Paradoxo/contradição
- Risco pessoal
- Injustiça revelada
- Descoberta inesperada

### Como Aplicar (Técnica, não Estilo)

**Antes (sem tensão):**
```
O problema é a inflamação intestinal.
A solução é este suplemento.
```

**Depois (com tensão):**
```
O problema não é falta de força de vontade.
É uma toxina invisível que sequestra seu apetite.
E ninguém te contou isso porque... [TENSÃO: por quê esconderam?]
```

### Teste Universal
> "Existe algo em jogo além do problema técnico? O leitor SENTE conflito?"

---

## 2. Descoberta em Tempo Real

### O que é
O avatar parece estar PENSANDO enquanto fala, não APRESENTANDO algo pronto.

### Princípio Universal
A copy deve parecer reflexão em voz alta, não slide de PowerPoint.

### Como Aplicar (Técnica, não Estilo)

**Antes (apresentação):**
```
O método funciona em 3 etapas.
Primeiro, você faz X.
Segundo, você faz Y.
Terceiro, você faz Z.
```

**Depois (descoberta):**
```
Aí eu pensei: se existe há 15 anos, alguém já mapeou isso.
Fui procurar e achei...
Na primeira vez que usei, parecia trapaça.
```

### Estrutura Universal
1. OBSERVAÇÃO → "Vi que todas as..."
2. QUESTIONAMENTO → "Mas por que funciona?"
3. REALIZAÇÃO → "Ah, é porque..."
4. IMPLICAÇÃO → "Isso significa que..."

### Teste Universal
> "Parece que o avatar está pensando ou apresentando?"

## 3. Transições Invisíveis

### O que é
Cada frase PUXA a próxima naturalmente. Não há "blocos" visíveis.

### Princípio Universal
Se você vê [IDENTIFICAÇÃO] → [MUP] → [CTA], a estrutura está VISÍVEL.
Se você lê como conversa fluida, a estrutura está INVISÍVEL.

### Como Aplicar (Técnica, não Estilo)

**Antes (estrutura visível):**
```
[IDENTIFICAÇÃO]
Você já tentou emagrecer e não conseguiu.

[MUP]
O problema é a inflamação intestinal.

[SOLUÇÃO]
Este método resolve.
```

**Depois (transição invisível):**
```
Você já tentou emagrecer e não conseguiu?
Talvez você não saiba, mas o problema não é falta de força de vontade.
É uma coisa que ninguém te contou...
```

### Conectores Naturais
- "Talvez você não saiba, mas..."
- "E sabe por quê?"
- "Aí eu pensei:"
- "Foi então que..."
- "O que me deixa [emoção] é..."

### Teste Universal
> "Cubra a próxima frase. Você SENTE falta dela? A anterior pede continuação?"

## 4. Especificidade = Cena de Filme (v5.0 - ATUALIZADO)

### O que é
Descrições com DUAS FACES: DATA e NARRATIVA. Narrativa é mais poderosa.

### Princípio Universal v5
Uma memória específica > um número estatístico.
NARRATIVA > DATA em poder emocional.
"Me custou a vaga" > "perdi 10 pontos".

### Como Aplicar (Técnica, não Estilo)

**FACE DATA (números soltos):**
```
Estudei 4 anos.
89% dos candidatos reprovam.
Perdi 12kg em 47 dias.
```

**FACE NARRATIVA (cena de filme - MAIS PODEROSA):**
```
Em 6 anos pagando cursinho, ninguém nunca me ensinou isso.
Uma palavra. Uma única palavra me custou a vaga na Polícia Federal.
Minha esposa achou que eu estava doente quando viu.
```

### Framework de Transformação v5
| De (DATA) | Para (NARRATIVA) |
|-----------|------------------|
| Número | Consequência pessoal |
| Tempo | Oportunidade perdida |
| Estatística | Memória específica |
| "12kg" | "Calça que não servia há 3 anos" |

### Teste Universal v5
> "Consigo VER O FILME? É dado ou é cena?"
> "É estatística ou é memória dolorosa?"

## 5. Ritmo/Respiração

### O que é
Variação natural de frases que permite o leitor "respirar".

### Princípio Universal
Não é sobre regras de tamanho (máx 12 palavras). É sobre variação NATURAL.

### Como Aplicar (Técnica, não Estilo)

**Antes (monótono):**
```
Você estuda todos os dias. Você faz questões todos os dias.
Você revisa todos os dias. Mas não passa.
```

**Depois (respiração):**
```
Você estuda. Estuda. Estuda.
E na hora da prova?
Erra o que sabia.
```

### Padrão de Respiração
```
Curta.
Curta.
Curta.
Média com informação nova.
PUNCH que dói.
[Pausa - frase longa reflexiva que dá descanso]
```

### Teste Universal
> "LER EM VOZ ALTA. Onde você respira naturalmente? Existe variação?"

## 6. Tom Conversacional (Base) - Tom Específico (Calibrado)

### O que é
**BASE (universal):** Escrever como conversa, não como documento.
**TOM ESPECÍFICO (calibrado):** Raiva, vulnerabilidade, autoridade → vem do BRIEFING e DRE.

### Princípio Universal v5
- **Tom conversacional** = FUNDAMENTO (sempre aplicar)
- **Tom emocional específico** = CALIBRADO por briefing (não fixo)
- DRE define espectro: vulnerabilidade ↔ raiva

### Como Aplicar (Técnica, não Estilo)

**Antes (documento):**
```
É importante ressaltar que a metodologia apresentada
demonstra resultados significativos em diversos estudos.
```

**Depois (conversacional - BASE):**
```
E sabe por quê funciona?
Porque não é sobre estudar mais. É sobre estudar certo.
```

**Depois + Tom Calibrado (DRE = Raiva):**
```
Sabe por que ninguém te contou isso?
Porque cursinho ganha quando você reprova. De novo. E de novo.
```

### Indicadores de Tom Conversacional (BASE)
- Perguntas retóricas
- Frases incompletas (pontos de suspensão)
- Contrações naturais
- Expressões de emoção

### Tom Específico (CALIBRADO - NÃO UNIVERSAL)
**Ver briefing para:**
- DRE do público (medo, raiva, desejo, etc.)
- Espectro tonal definido
- Swipes do nicho

### Teste Universal
> "Parece conversa ou documento?" (BASE)
> "O tom está calibrado com a DRE do briefing?" (ESPECÍFICO)

## Checklist de Fundamentos

Antes de entregar qualquer copy, validar:

- [ ] **Tensão:** Existe conflito emocional? Algo em jogo?
- [ ] **Descoberta:** Parece pensamento ou apresentação?
- [ ] **Transições:** Cada frase puxa a próxima?
- [ ] **Especificidade:** Memória dolorosa ou número solto?
- [ ] **Ritmo:** Variação natural? Pausas para respirar?
- [ ] **Conversacional:** WhatsApp ou documento?
