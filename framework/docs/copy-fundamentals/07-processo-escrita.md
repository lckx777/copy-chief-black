# Fundamento 07: Processo Profissional de Escrita

> O workflow que separa amadores de profissionais — COPYWORK → DRAFT → INCUBATE → EDIT.

---

## As 4 Etapas

```
COPYWORK → DRAFT → INCUBATE → EDIT
     ↓         ↓         ↓         ↓
Aquecimento  Rascunho  Descanso  Refinamento
  (absorve)  (produz)  (processa)  (lapida)
```

---

## Etapa 1: COPYWORK (Aquecimento)

### O que é
Copiar À MÃO textos que você admira por 10-15 minutos.

### Por que funciona
- Absorve o estilo por osmose
- Aquece o "músculo" da escrita
- Internaliza padrões de qualidade
- Prepara a mente para produção

### Como Fazer

1. Escolher 1-2 textos do swipe file
2. Copiar À MÃO (não digitar)
3. 10-15 minutos antes de escrever
4. Focar em textos relevantes ao que vai produzir

### O que Copiar

| Tipo de Copy | O que Copiar |
|--------------|--------------|
| VSL | Leads de VSLs validadas |
| LP | Headlines e bullets |
| Email | Subject lines + aberturas |
| Social | Hooks de criativos escalados |

---

## Etapa 2: DRAFT (Rascunho Rápido)

### O que é
Escrever o mais rápido possível, sem julgamento.

### Princípios

- **Velocidade sobre qualidade** — primeiro draft
- **Aceitar que vai parecer ruim** — normal
- **NÃO editar ainda** — apenas produzir
- **Seguir o fluxo** — não parar para corrigir

### Como Fazer

1. Definir estrutura macro (outline)
2. Timer: 25-50 min sem parar
3. Escrever sem revisar
4. Aceitar imperfeições
5. Completar até o fim

### Mentalidade

```
"O primeiro draft de qualquer coisa é lixo."
— Ernest Hemingway

"Não consigo escrever cinco palavras sem mudar sete."
— Dorothy Parker
```

**Liberte-se:** O draft ruim é o primeiro passo para o texto bom.

---

## Etapa 3: INCUBATE (Incubação)

### O que é
Afastar-se do texto e deixar o cérebro processar.

### Por que funciona

- Processamento passivo continua
- Insights surgem "do nada"
- Distância cria perspectiva
- Erros ficam mais visíveis depois

### Quanto Tempo

| Tipo de Copy | Tempo de Incubação |
|--------------|-------------------|
| Email/Social | 1-4 horas |
| LP sections | 4-24 horas |
| VSL completa | 24-48 horas |
| Projeto grande | 48-72 horas |

### O que Fazer Durante

- Não pensar ativamente no texto
- Fazer outras atividades
- Deixar mente vagar
- Anotar insights quando surgirem

---

## Etapa 4: EDIT (Edição)

### O que é
Onde a grandeza acontece — refinar, cortar, polir.

> "Sou um escritor medíocre, mas sou um bom editor."
> — David Ogilvy

### Princípios

- **Kill your darlings** — cortar sem dó
- **Menos é mais** — cada palavra deve justificar
- **Ritmo** — variar comprimento de frases
- **Clareza** — se confunde, simplificar

### Checklist de Edição

**Cortes:**
- [ ] Cortei palavras desnecessárias?
- [ ] Cortei parágrafos que não agregam?
- [ ] Cortei repetições?
- [ ] Cortei frases que não passam no Hand Test?

**Clareza:**
- [ ] Cada frase tem uma ideia só?
- [ ] Linguagem é de conversa?
- [ ] Leitor de 12 anos entenderia?
- [ ] Não há jargão desnecessário?

**Ritmo:**
- [ ] Há variedade de comprimento?
- [ ] Frases curtas para impacto?
- [ ] Frases longas para emoção?

**Final:**
- [ ] Lido em voz alta soa natural?
- [ ] Flui de frase em frase?
- [ ] CTA é claro?

---

## Erro Fatal

### O que NÃO fazer

❌ **Tentar DRAFT + EDIT na mesma sessão**

### Por que é fatal

- Modo criativo vs modo crítico são opostos
- Alternar trava o fluxo
- Produção fica lenta
- Qualidade sofre em ambos

### A Regra

```
SEPARAR CRIAÇÃO DE EDIÇÃO.
Primeiro produz.
Depois incuba.
Depois edita.
```

## Workflow Completo

### Sessão de Trabalho Ideal

```
MANHÃ (2-3h):
├── 10-15 min: Copywork (aquecimento)
├── 45-60 min: Draft 1 (produção)
├── 15 min: Pausa
└── 45-60 min: Draft 2 (outra seção)

TARDE/NOITE (após incubação):
├── 30-60 min: Edição Draft 1
└── 30-60 min: Edição Draft 2
```

### Para Projetos Grandes

| Dia | Atividade |
|-----|-----------|
| Dia 1 | Copywork + Draft completo |
| Dia 2 | Incubação (fazer outras coisas) |
| Dia 3 | Primeira edição |
| Dia 4 | Segunda edição + finalização |

## Ferramentas de Apoio

### Para Draft
- Foco: Bloqueador de distrações
- Timer: Pomodoro (25min)
- Ambiente: Sem notificações

### Para Edição
- Hemingway App: Nível de leitura
- Leitura em voz alta: Ritmo
- Print físico: Erros saltam mais

### Swipe File
- Organizado por tipo de copy
- Anotado com aprendizados
- Atualizado regularmente

---

## Checklist de Processo

### Antes de Escrever
- [ ] Fiz copywork de aquecimento?
- [ ] Defini estrutura/outline?
- [ ] Ambiente sem distrações?

### Durante Draft
- [ ] Estou escrevendo sem parar?
- [ ] Não estou editando?
- [ ] Aceito imperfeições?

### Durante Incubação
- [ ] Afastei do texto?
- [ ] Tempo adequado passou?
- [ ] Anotei insights que surgiram?

### Durante Edição
- [ ] Cortei o desnecessário?
- [ ] Cada frase justifica?
- [ ] Lido em voz alta?
- [ ] Ritmo variado?
