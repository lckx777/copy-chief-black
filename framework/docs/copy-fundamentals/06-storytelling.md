# Fundamento 06: Storytelling Persuasivo

> Intenção, obstáculo e ritmo — os elementos de narrativas que vendem.

---

## Princípio de Aaron Sorkin

> "Eu venero no altar de INTENÇÃO e OBSTÁCULO."

Toda história persuasiva tem dois elementos essenciais:

| Elemento | Definição | Função |
|----------|-----------|--------|
| **INTENÇÃO** | O que o personagem quer | Cria identificação e desejo |
| **OBSTÁCULO** | O que está no caminho | Cria tensão e engajamento |
| **STAKES** | O que está em jogo | Eleva importância emocional |

---

## Teste de Storytelling

> "Pause em qualquer momento da sua copy. Você sabe o que o personagem quer e o que impede? Se não, reescreva."

### Exemplo: Copy de Emagrecimento

```
INTENÇÃO: Mulher quer emagrecer para caber no vestido do casamento
OBSTÁCULO: Toxina intestinal bloqueia metabolismo
STAKES: Casamento da filha em 60 dias
```

**Copy derivada:**
> "Maria tinha 60 dias para caber naquele vestido.
> Não era qualquer vestido — era o casamento da filha.
> Mas nada funcionava. Dietas, academia, remédios.
> Até que descobriu o que estava travando tudo..."

## High Stakes in Low Stakes

### O Conceito

A arte é criar **ALTAS APOSTAS EMOCIONAIS** em **SITUAÇÕES DE BAIXA APOSTA**.

| Situação | Aposta Real | Aposta Emocional |
|----------|-------------|------------------|
| Cortaram a fila | Baixa | Indignação intensa |
| Não coube na calça | Baixa | Vergonha profunda |
| Reprovado pela 6ª vez | Média | Questionamento existencial |

### Por que Funciona

- Histórias relacionáveis multiplicam impacto
- Avatar se vê na situação
- Emoção amplificada por identificação

### Aplicação

```
Errado: "Ela estava frustrada com o peso."
→ Genérico, não gera identificação

Certo: "Ela olhou no espelho do banheiro e não se reconheceu.
A mesma mulher que há 5 anos usava biquíni na praia.
Agora não conseguia nem sair de casa no fim de semana."
→ Situação específica, emoção palpável
```

## Estrutura de História Persuasiva

### Arco Narrativo

```
1. SITUAÇÃO INICIAL (Identificação)
   → Avatar estava assim, como você

2. EVENTO DISPARADOR (Problema)
   → Algo aconteceu que mudou tudo

3. LUTA (Obstáculos)
   → Tentou isso, aquilo, nada funcionou

4. DESCOBERTA (Virada)
   → Até que descobriu [mecanismo]

5. TRANSFORMAÇÃO (Resultado)
   → E agora está assim

6. CHAMADA (CTA)
   → Você pode fazer o mesmo
```

### Exemplo Completo

```
SITUAÇÃO: "Eu era como você. Estudava 10h por dia, fazia cursinho caro..."

EVENTO: "Aí veio a 6ª reprovação. A mais dolorosa."

LUTA: "Tentei mudar de material. De professor. De rotina.
Nada mudava o resultado."

DESCOBERTA: "Até que um aprovado me mostrou algo diferente.
Não era sobre estudar mais. Era sobre estudar certo."

TRANSFORMAÇÃO: "3 meses depois, meu nome estava na lista.
Não acreditei quando vi."

CHAMADA: "Se você está onde eu estava,
precisa ver isso."
```

## Ritmo Narrativo (Great Writing Has Rhythm)

### O Erro
Todas as frases do mesmo tamanho = monotonia.

### A Solução
Variar comprimento para criar "música".

```
Esta frase tem cinco palavras.
Aqui estão mais cinco palavras.
Frases curtas são ok.
Mas várias juntas ficam monótonas.
O ouvido cansa.

Agora ouça.
Eu vario o comprimento e crio música.
A escrita canta.
Tem ritmo, melodia, harmonia.

Uso frases curtas.
E uso médias.
E às vezes, quando o leitor está descansado,
eu o envolvo com uma frase longa que queima com energia
e cresce como um crescendo — sons que dizem:
ouça isso, isso é importante.
```

### Regra Prática

| Tipo | Quando Usar |
|------|-------------|
| **Curta** (1-5 palavras) | Punch, impacto, transição |
| **Média** (6-12 palavras) | Explicação, contexto |
| **Longa** (13+ palavras) | Clímax, emoção intensa |

**Padrão:** Curto por padrão. Longo para impacto quando leitor está "descansado".

## Elementos de Tensão

### Como Criar Tensão

| Técnica | Exemplo |
|---------|---------|
| **Conflito** | Protagonista vs Antagonista |
| **Tempo** | Deadline se aproximando |
| **Risco** | O que pode dar errado |
| **Incerteza** | Não saber se vai funcionar |
| **Stakes** | O que está em jogo se falhar |

### Antagonistas por Nicho

| Nicho | Antagonista |
|-------|-------------|
| Saúde | Indústria farmacêutica |
| Finanças | Bancos / Governo |
| Concursos | Cursinhos caros |
| Relacionamento | Coaches superficiais |
| Carreira | Chefes / Sistema corporativo |

## Checklist de Validação

### Estrutura
- [ ] Intenção clara do protagonista?
- [ ] Obstáculo definido?
- [ ] Stakes elevados?
- [ ] Arco completo (início-meio-fim)?

### Identificação
- [ ] Avatar se vê na história?
- [ ] Situações são específicas?
- [ ] Emoções são palpáveis?

### Ritmo
- [ ] Variedade de comprimento?
- [ ] Frases curtas para impacto?
- [ ] Frases longas para clímax?

### Tensão
- [ ] Há conflito?
- [ ] Há incerteza?
- [ ] Stakes são claros?

---

## Erros Comuns

| Erro | Consequência | Correção |
|------|--------------|----------|
| História sem obstáculo | Não gera tensão | Adicionar antagonista/conflito |
| Stakes baixos | Não importa para o leitor | Elevar consequências |
| Frases monótonas | Cansa, perde atenção | Variar ritmo |
| Protagonista genérico | Não gera identificação | Detalhes específicos |
| Arco incompleto | Insatisfação | Garantir resolução |

## Templates de História

### Template 1: Jornada do Herói Comprimida
```
"[Personagem] era [situação inicial].
Um dia, [evento disparador].
Tentou [obstáculo 1], [obstáculo 2], [obstáculo 3].
Nada funcionava.
Até que [descoberta].
Hoje, [transformação].
Se você [identificação], [CTA]."
```

### Template 2: Antes/Depois Emocional
```
"Antes: [Situação dolorosa com detalhes sensoriais]
O momento que mudou tudo: [Descoberta]
Depois: [Nova realidade com detalhes sensoriais]"
```

### Template 3: Conflito com Sistema
```
"[Eles] não querem que você saiba disso.
Por quê? Porque [razão do conflito].
Mas eu descobri [verdade oculta].
E agora você pode [benefício]."
```
