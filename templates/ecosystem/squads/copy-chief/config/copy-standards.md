# Copy Standards — Copy Chief BLACK

## DRE-First
- Dominant Resident Emotion drives ALL copy decisions
- DRE identified from VOC data, never assumed
- Escalation: Surface → Personal → Relational → Identity → Existential
- Copy must reach level 4-5 at action points

## Anti-Homogenization
- Logo Test: competitor CANNOT use without changing
- Zero banned words (see domains/anti-homog)
- Zero niche cliches
- Avatar's exact language from VOC (not paraphrase)
- Sensory future pacing (sight + sound + physical)

## Visceral > Logical
- Absolute language, zero hesitation
- Fragments > complete sentences (when impact is greater)
- Conversation > formal writing
- Body reaction > mind understanding

## Copy in Files
- NEVER output copy to terminal/chat
- All copy → production/{deliverable}/ directories
- Attribution header in every file

## Quality Thresholds
- blind_critic >= 8 per unit
- emotional_stress_test genericidade >= 8
- black_validation >= 8 before delivery
- Logo Test PASS
- Mecanismo VALIDATED before production
