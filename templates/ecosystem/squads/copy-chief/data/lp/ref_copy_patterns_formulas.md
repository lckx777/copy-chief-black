# Copy Patterns e Formulas Avancadas

Patterns de alta conversão para elevação de copy.

---

## FORMULAS DE HEADLINE

### Formula 1: Padrao (Promessa + Tempo + Objecao)

```
Descubra como {PROMESSA} em {TEMPO}, mesmo que {OBJEÇÃO}.
```

### Formula 2: Pergunta Direta

```
Você quer {PROMESSA} em {TEMPO}?
Mesmo que você {OBJEÇÃO}?
Então preste atenção...
```

### Formula 3: Comando + Resultado

```
{AÇÃO IMPERATIVA} e {RESULTADO} em {TEMPO}.
Funciona mesmo que você {OBJEÇÃO}.
```

### Formula 4: Revelacao

```
A verdade sobre {PROBLEMA} que ninguém te conta...
E como {PROMESSA} em {TEMPO}.
```

### Formula 5: Prova Social Integrada

```
Como {QUANTIDADE} pessoas já {RESULTADO} em {TEMPO}...
E como você pode fazer o mesmo, mesmo que {OBJEÇÃO}.
```

## PATTERNS DE RUMINACAO

### Pattern 1: Espiral Descendente

```
Você não aguenta mais {DOR 1}.
E quando {DOR 2}, você sente {EMOÇÃO}.
Por isso pensa: "{RUMINAÇÃO 1}"
E também: "{RUMINAÇÃO 2}"
```

Função: Intensifica dor progressivamente.

### Pattern 2: Validacao Empatica

```
Eu sei como é.
Eu sei que você {DOR ESPECÍFICA}.
E sei que você já {TENTATIVA FALHA}.
Estou certo?
```

Função: Cria conexão através de compreensão.

### Pattern 3: Ruminacoes em Aspas

```
- "Por que {PERGUNTA INTERNA 1}?"
- "Será que {DÚVIDA}?"
- "{PENSAMENTO DE FRUSTRAÇÃO}"
- "Só queria {DESEJO SIMPLES}..."
```

Função: Captura diálogo interno do avatar.

### Pattern 4: Transicao Condicional

```
Agora eu te pergunto...
Se você pudesse {RESULTADO DESEJADO},
sem {OBJEÇÃO ELIMINADA},
isso mudaria sua vida?

Se SIM, o {PRODUTO} é para você.
```

Função: Cria comprometimento condicional.

## PATTERNS DE ENTREGAVEIS

### Pattern 1: Checklist

```
CHECKLIST {RESULTADO}

Você terá em mãos um checklist de {TIPO} ações
que tornará impossível você não {RESULTADO}
em {TEMPO}.
```

### Pattern 2: Template Pratico

```
TEMPLATE DE {APLICAÇÃO}

Guia completo com {CARACTERÍSTICA} exercícios
para você aplicar {TEMPO} e {BENEFÍCIO},
sem precisar {OBJEÇÃO}.
```

### Pattern 3: Guia de Acoes

```
GUIA DE AÇÕES PARA {RESULTADO}

Um passo a passo para você {AÇÃO},
ou seja, {EXPLICAÇÃO}.
Você vai saber exatamente como {TRANSFORMAÇÃO}.
```

### Pattern 4: Lista de Erros

```
LISTA DE ERROS QUE TE IMPEDEM DE {RESULTADO}

Você vai receber uma lista com os {NÚMERO} erros
mais comuns que acontecem quando {SITUAÇÃO}.
```

## PATTERNS DE ANCORAGEM

### Pattern 1: Stack de Valor

```
- {ENTREGÁVEL 1} .... R$ {VALOR 1}
- {ENTREGÁVEL 2} .... R$ {VALOR 2}
- {BÔNUS 1} ......... R$ {VALOR 3}
- {BÔNUS 2} ......... R$ {VALOR 4}

TOTAL: R$ {SOMA}

Por apenas: R$ {PREÇO REAL}
```

Regra: SOMA = 3x a 5x do PREÇO REAL.

### Pattern 2: Comparacao Externa

```
Uma sessão de {ESPECIALISTA} custa R$ {VALOR ALTO}.
Com o {PRODUTO}, você tem {EQUIVALÊNCIA}
por apenas R$ {PREÇO}.
```

### Pattern 3: Custo Diario

```
Por menos de R$ {PREÇO/30} por dia,
você terá acesso a {BENEFÍCIO PRINCIPAL}.
Menos que {COMPARAÇÃO COTIDIANA}.
```

## PATTERNS DE CTA

### Pattern 1: Padrao com Urgencia

```
{PARCELAMENTO}x de R$ {VALOR}
ou R$ {À VISTA}

ACESSO ILIMITADO

[SIM, QUERO APROVEITAR]

Compra segura | Acesso imediato
```

### Pattern 2: Com Escassez

```
{PARCELAMENTO}x de R$ {VALOR}

OFERTA POR TEMPO LIMITADO
Restam {NÚMERO} vagas

[GARANTIR MINHA VAGA AGORA]
```

### Pattern 3: Com Garantia Destacada

```
{PARCELAMENTO}x de R$ {VALOR}

GARANTIA DE {DIAS} DIAS
Se não funcionar, devolvemos seu dinheiro.

[QUERO TESTAR SEM RISCO]
```

## PATTERNS DE CONVERSA SERIA

### Pattern 1: Escolha Binaria Classica

```
AGORA VOCÊ TEM 2 ESCOLHAS

Opção 1: Continuar {DOR}

Ou

Opção 2: {TRANSFORMAÇÃO}

A escolha é sua.
Mas eu sei qual é a mais inteligente.
```

### Pattern 2: Futuro Projetado

```
Daqui a {TEMPO}, você vai estar em um de dois lugares:

1. No mesmo lugar de hoje, {DOR MANTIDA}

2. {RESULTADO ALCANÇADO}, celebrando sua {CONQUISTA}

Qual futuro você escolhe?
```

### Pattern 3: Custo da Inacao

```
Você pode continuar fazendo o que sempre fez.
E continuar tendo os mesmos resultados.

Ou pode {AÇÃO} hoje
e finalmente {TRANSFORMAÇÃO}.

Cada dia que passa é um dia a menos de {BENEFÍCIO}.
```

## REGRAS DE CONSISTENCIA

### Promessa Principal

Deve aparecer identica em:
- Bloco 1 (Headline)
- Bloco 3 (Transição da ruminação)
- Bloco 7 (Fechamento "para quem serve")
- Bloco 9 (CTA)

### DRE (Dominant Resident Emotion)

Deve aparecer em:
- Bloco 3 (Ruminações)
- Bloco 10 (Conversa séria - Opção 1)

### Tempo Especifico

Deve aparecer em:
- Bloco 1 (Headline)
- Bloco 5 (Entregáveis)
- Bloco 6 (Bônus)

### Preco

Deve ser consistente em:
- Bloco 8 (Recapitulando)
- Bloco 9 (CTA)
- Bloco 12 (CTA repetido)
