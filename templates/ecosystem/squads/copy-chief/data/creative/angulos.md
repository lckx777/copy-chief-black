## Super Estrutura do Mecanismo Unico

### 4 Camadas

| Camada | Descricao | Exemplo |
|--------|-----------|---------|
| Ingrediente Hero | Resolve a causa | Berberina, Sal Rosa |
| Gimmick Name | Nome "chiclete" | Pink Salt Trick, Ovelha Negra |
| Origin Story | De onde veio | "Receita de atrizes de Hollywood" |
| Authority Hook | Gancho de autoridade | "Ozempic Natural", "Truque do Cavalo" |

### Formula do Gancho

```
"Ja ouviu falar desse [GIMMICK NAME] que [ORIGEM] estao usando para [DESEJO]? Chamam de [AUTHORITY HOOK]."
```

Gimmick Name gera curiosidade misturada com prova — irresistivel.

## Regras

1. Hook curto, forte, direto
2. Combinar angulos livremente
3. Impacto emocional obrigatorio
4. Paradoxo gera curiosidade
5. Gimmick Name > nome generico
